# feishu-doc-parser
Feishu doc Python parser toolkit — parse_all.py / extract_all.py / login automation.

## Tools
- **parse_all.py** — Block-level Feishu doc API parser, supports 13 block types
- **extract_all.py** — Batch extraction from API response files
- **feishu_login_auto.js** — Feishu login page automation script
